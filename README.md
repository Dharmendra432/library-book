## 📌 Project Overview

The **Online Library System** is a modern web application built using **React (with Vite)** that allows users to browse, search, and manage books efficiently.

It demonstrates core frontend concepts like **routing, state management, and dynamic UI rendering**.

---

## 🚀 Features

* 🏠 Home page with book categories and popular books
* 📚 Browse books by category using dynamic routing
* 🔍 Search functionality (filter by title or author)
* 📖 Detailed book view page
* ➕ Add new books using a form
* 🧠 State management using Redux Toolkit
* 🚫 404 Page for invalid routes
* 🎨 Responsive and clean UI design

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🔁 React Router DOM
* 🧠 Redux Toolkit
* 🎨 CSS (Custom Styling)

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Dharmendra432/library-system.git

# Navigate into the project folder
cd library-system

# Install dependencies
npm install

# Run the development server
npm run dev
```
